Developed Concentration, a Chrome extension that auto-blocks distracting websites based on user inputs using NLP, LLMs, a Flask backend, and Firebase. The project reached 2.2K+ users, earned a 4.6/5 rating on the Chrome Web Store, and was connected to provisional patent application 63/383,080.
